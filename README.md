# SolNAS Client

#### The sister application to the standalone program SolNAS.

(Get ready to read)

SolNAS is supposed to be very simple. Either extract the release and run the program, or run it from your terminal, or build your own executible file using rust. I don't mind.

## Features

- Remotely change the config of SolNAS
- Customizable UI
- Easy Dashboard Management
- Multi-upload / Download
- Improved User Control
- Come on, it's better in every way.

# Customizing UI

- A file called style.ini is generated when running the client. 
- The colors of each of the buttons on the dashboard are controlled here
- The colors are in hex. 

Google 'hex color' , choose your colors, copy and paste. Voila.

# Important Note:

- SolNAS will tell you your IP when you start up the program. Use that IP when trying to log in.
